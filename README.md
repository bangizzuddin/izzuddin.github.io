# Ahmad Izzuddin Faqih Portfolio Website

This is a static, responsive portfolio website.

## Files
- `index.html` — main website page
- `styles.css` — professional responsive styling
- `assets/` — profile and portfolio images

## How to use
Open `index.html` directly in a browser, or upload the whole folder to a static hosting service such as Netlify, Vercel, GitHub Pages, or your own web hosting.
